How to create YOUR very own website FOR FREE!

Step 1: Paste this code into a blank text file:

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
</body>
</html>

Step 2: Change the text file's name from index.txt to index.html

Step 3: To see your website, simply drag and drop index.html into any web browser!
